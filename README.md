# Data-Visualization-project how to run


# Without virtual environment:

open folder in cmd

pip install -r requirements.txt

py gameSales.py

go to http://127.0.0.1:8050/ in browser

# With virtual environment (recommended):

open folder in cmd

py -m venv venv

venv\scripts\activate

pip install -r requirements.txt

py gameSales.py

go to http://127.0.0.1:8050/ in browser
